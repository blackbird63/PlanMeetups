# Plan Meetups built with React & NextJS 

## Available Commands

| Command         | Description                                                                                                                     |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| `npm i`         | Install project dependencies                                                                                                    |
| `npm run dev`   | Builds project and open web server (with Vite)                                                             |
| `npm run build` | Builds code bundle with production settings
